# Installing beta SDKs

To use features like [sandbox persistence](/docs/sandbox/persistence) you need to install the beta version of the SDKs.

### Code Interpreter SDK

<CodeGroup>
```bash {{ language: 'js' }}
npm i @e2b/code-interpreter@beta
```

```bash {{ language: 'python' }}
pip install e2b-code-interpreter==1.2.0b5
```
</CodeGroup>

### Core SDK

<CodeGroup>
```bash {{ language: 'js' }}
npm i e2b@beta
```

```bash {{ language: 'python' }}
pip install e2b==1.2.0b6
```
</CodeGroup>

### Desktop SDK

<CodeGroup>
```bash {{ language: 'js' }}
npm i @e2b/desktop@beta
```

```bash {{ language: 'python' }}
pip install e2b-desktop==1.7.1b1
```
</CodeGroup>
