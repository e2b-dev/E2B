title: "Create charts & visualizations"
---

E2B Sandbox allows you to create charts and visualizations by executing Python code inside the sandbox with `runCode()` method in JavaScript and `run_code()` method in Python.

These charts and visualizations can be [static](/code-interpreting/create-charts-visualizations/static-charts) or [interactive](/code-interpreting/create-charts-visualizations/interactive-charts) plots.
