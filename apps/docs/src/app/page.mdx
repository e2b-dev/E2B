# What is E2B?

**Sandboxed cloud environments for AI agents & AI apps {{ className: 'lead' }}**

With E2B, you can create AI playgrounds. In these AI playgrounds, you can let your AI agents and apps:

- [Run any code](/agents/exec) and terminal command in a sandboxed environment
- [Start any process](/playgrounds/process) (even long-running such as a database)
- [Run any shell command](/agents/shell-commands)
- [Upload and download files](/playgrounds/filesystem#write-bytes)
- Install dependencies and programs
- [Use filesystem](/playgrounds/filesystem)
- Access the internet
- Start a web server that's accessible from the internet
- [Clone git repositories](/agents/clone-repo)
