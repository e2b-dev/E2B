# Runtime Sandbox for Large Language Models

Sandboxes allow AI apps to have long running cloud environments where large language models can use the same tools as human do. {{ className: 'lead' }}

## Use cases
Our Sandboxes are general purpose machines powered by Ubuntu. Most of the things your AI app or agent can do locally can also be done in our sandboxes.
Here's a few use cases:

- **AI data analysis**
- **Code interpreter**
- **Run LLM-generated code**
- **Coding copilots and agents**
- **AI powered browsers in cloud**

## Quick start

### 1. Create an account
<Button variant="textLink">Sign in</Button> with your GitHub acccount.

### 2. Get your API key
After sign in, get [your API key](/getting-started/api-key).

### 3. Install the E2B SDK
We offer both JS and Python SDK.

<CodeGroup isRunnable={false}>
```bash {{ language: 'js' }}
npm i @e2b/sdk
```

```bash {{ language: 'python' }}
pip install e2b
```
</CodeGroup>

### 4. Start up the sandbox
<CodeGroup isRunnable={false}>
```js {{ language: 'js' }}
// 1. Import E2B
import { Session } from '@e2b/sdk'

// 2. Get your Sandbox session
const session = await Session({
  // or 'Bash', 'Python3', 'Java', 'Go', 'Rust', 'PHP', 'Perl', 'DotNET'
  id: 'Nodejs',
})

// 3. Close the session once done
await session.close()
```

```python {{ language: 'python' }}
# 1. Import E2B
from e2b import Session

# 2. Get your Sandbox session
session = Session(
  # or "Bash", "Python3", "Java", "Go", "Rust", "PHP", "Perl", "DotNET"
  id: "Nodejs",
)

# 3. Close the session once done
session.close()
```
</CodeGroup>
