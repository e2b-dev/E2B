# Sandbox for AI Apps & Apps

E2B Sandbox is a secure sandboxed cloud environment made for AI agents and AI apps.
Sandboxes allow AI agents and apps to have long running cloud secure environments. In these environments, large language models can use the same tools as humans do. {{ className: 'lead' }}

## Use cases
Under the hood, the sandbox is a general purpose machines powered by Ubuntu. Most of the things your AI app or agent can do locally can also be done in our sandboxes.
Here are a few use cases:

- **AI data analysis**
- **Code interpreter**
- **Run LLM-generated code**
- **Coding copilots and agents**
- **AI powered browsers in the cloud**

## Quick start

### 1. Create an account
<Button variant="textLink">Sign in</Button> with your GitHub acccount.

### 2. Get your API key
After sign in, get [your API key](/getting-started/api-key) and set it in the `E2B_API_KEY` environment variable.

### 3. Install the E2B SDK
We offer both JS and Python SDK.

<CodeGroup isRunnable={false}>
```bash {{ language: 'js' }}
npm install @e2b/sdk
```

```bash {{ language: 'python' }}
pip install e2b
```
</CodeGroup>

### 4. Start up the sandbox
Sandbox is an isolated cloud environment for your AI app, agent, assistant, etc.

<CodeGroup isRunnable={false}>
```js {{ language: 'js' }}
// 1. Import E2B
import { Sandbox } from '@e2b/sdk'

// 2. Get your Sandbox session
const sandbox = await Sandbox.create()

// 3. Close the sandbox once done
await sandbox.close()
```

```python {{ language: 'python' }}
# 1. Import E2B
from e2b import Sandbox

# 2. Get your Sandbox session
sandbox = Sandbox()

# 3. Close the sandbox once done
sandbox.close()
```
</CodeGroup>

By default the `base` sandbox supports NodeJS 20 and Python 3.10. If you want customize your sandbox (copy files, install dependencies, etc), check out [Sandbox Templates](/sandbox/templates/custom).
