# Use premade data analysis sandbox template

You can use our premade template to get started with your data analysis project.


### Using data analysis template in your code

<CodeGroup isRunnable={false}>
```js {{ language: 'js' }}
import { Sandbox } from '@e2b/sdk'

const sandbox = await Sandbox.create({
  id: 'Python3-DataAnalysis',
})

await sandbox.close()
```

```python {{ language: 'python' }}
from e2b import Sandbox

sandbox = Sandbox(
  id: "Python3-DataAnalysis",
)

sandbox.close()
```
</CodeGroup>

The Dockerfile and template files used to build the template are available [here](https://github.com/e2b-dev/e2b/blob/main/templates/data_analysis/).
