# Sandboxes Templates

Sandbox is a cloud environment for your AI agent or AI app.

You can customize sandboxes by creating a sandbox template or using one of our premade sandbox templates.

### Creating custom sandbox templates
Check out how to create custom sandbox templates [here](/sandbox/templats/custom).

### Premade sandbox templates
We have the following premade sandbox templates:
- `base` - template with NodeJS 20 and Python 3.10
- [`Python3-DataAnalysis`](/sandbox/templates/data-analysis) - template with Python 3.10 and data analysis libraries


### Using sandbox templates in your code

<CodeGroup isRunnable={false}>
```js {{ language: 'js' }}
import { Sandbox } from '@e2b/sdk'

const sandbox = await Sandbox.create({
  id: '<custom-or-premade-sandbox-template-id>',
})

await sandbox.close()
```

```python {{ language: 'python' }}
from e2b import Sandbox

sandbox = Sandbox(
  id: "<custom-or-premade-sandbox-template-id>",
)

sandbox.close()
```
</CodeGroup>
